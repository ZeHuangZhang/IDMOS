>  # IDMOS

   基于.NET的低代码开发平台和运维中心,集成管理后台、开放平台、服务监控、日志管理、运维管理等一系列功能,致力于为用户提供一站式的解决方案。

*  技术栈
		
	.NET8、Vue

*  终端
	
	 PC、H5

*  功能
	
	 多语言、认证中心、管理后台、组织管理、系统管理、 权限管理、 开放平台、 实时监控、  日志管理等。

> # 官网

[IDMOS](http://software.idmos.net)
